# 251CTT027_CaoNhatNguyen_ThiTKWebsite
Bài Thi Thiết kế web Mã sinh viên: 22123456 Họ tên sinh viên: Nguyễn Văn A  
Mô tả chức năng đã thực hiện:
​Thiết kế giao diện website Bảng điều khiển học tập cá nhân dành cho sinh viên bằng HTML5 và CSS3.
​Sử dụng cấu trúc thư mục chuẩn theo đúng yêu cầu phân tách các file index.html, css, js, images và docs.
​Xây dựng bố cục khoa học bằng cách kết hợp linh hoạt Flexbox (cho Header, Sidebar, Khu vực thống kê) và CSS Grid (cho Nội dung chính hiển thị các thẻ môn học).
​Hiển thị danh sách đầy đủ 6 thẻ môn học bao gồm: HTML/CSS, JavaScript, PHP, MySQL, Laravel và AI Cơ bản với đầy đủ hình ảnh, tiến độ (%) và nút chức năng.
​Xử lý sự kiện bằng JavaScript khi người dùng nhấn nút "Xem" tại mỗi thẻ môn học để cập nhật động dữ liệu (Tên môn học, Tiến độ) vào khu vực "MÔN HỌC ĐANG CHỌN".
​Cập nhật trạng thái kích hoạt (Active) trực quan cho các mục trên menu Sidebar bằng JavaScript khi người dùng nhấn chọn.
​Tự động tùy biến giao diện theo MSSV: Sử dụng JavaScript để tự động trích xuất 2 số cuối của MSSV khi tải trang, thực hiện kiểm tra chẵn/lẻ để thay đổi cấu hình bố cục Grid tương ứng (Grid 3 cột cho số chẵn và Grid 2 cột cho số lẻ).
​Áp dụng các hiệu ứng Hover và Transition mượt mà cho hệ thống các nút bấm, menu lựa chọn và các thẻ môn học nhằm tối ưu hóa trải nghiệm người dùng.
